    recap.paths

    recap.paths.CatalogPath ¶

    Bases: BaseModel

    Represents a path in the data catalog.

    template: ClassVar[str] class-attribute ¶

    A list of templated paths that represent the various locations of this path type. The template format uses Starlette's simplified URI style. For example:

    /databases/{scheme}/instances/{instance}/schemas/{schema}

    __str__() ¶

    Find a template format that has parameters compatible with this instance. If one is found, fill it in with this instance's attributes and return it. For example, a template:

    /databases/{scheme}/instances/{instance}/schemas/{schema}

    Would return:

    /databases/postgresql/instances/localhost/schemas/public

    If the instance had:

    SomeCatalogPath(
    scheme='postgresql',
    instance='localhost',
    schema='public']
)

    from_path(path) classmethod ¶

    Construct this model from a path if it matches the template.

    recap.paths.RootPath ¶

    Bases: CatalogPath

    Canonical root path for the catalog.

    recap.paths.create_catalog_path(path, *types) ¶

    Given a collection of CatalogPath types, find one that has a template that matches the provided path. If a matching path is found, instantiate the CatalogPath type and return it.
